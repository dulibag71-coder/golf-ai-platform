# ProView AI 랜딩 페이지 디자인 브레인스토밍

## 디자인 방향성 탐색

### 아이디어 1: "Cyberpunk Minimalism" (확률: 0.08)

**디자인 무브먼트**: 사이버펑크 미학과 미니멀리즘의 결합

**핵심 원칙**:
- 극도로 단순화된 형태와 기하학적 구조
- 네온 그린의 강렬한 대비를 통한 시각적 임팩트
- 디지털 시대의 정교함과 우아함 표현
- 공간의 여백을 적극 활용한 호흡감

**색상 철학**:
- 배경: Deep Charcoal (#121212) - 우주의 어둠, 무한한 가능성
- 주요 강조: Neon Green (#39FF14) - 디지털 생명력, AI의 활동성
- 보조 강조: 연두색 (#7FFF00) - 부드러운 에너지, 신뢰감
- 텍스트: 밝은 회색(#E8E8E8) - 높은 가독성, 차가운 톤

**레이아웃 패러다임**:
- 비대칭 구성: 왼쪽에 텍스트, 오른쪽에 동적 요소
- 그리드 기반이 아닌 자유로운 배치
- 수직 리듬감을 통한 스크롤 경험 강화
- 섹션 간 여백을 극대화하여 각 요소의 중요성 부각

**시그니처 요소**:
- 네온 그린의 얇은 선(border)으로 카드 강조
- 기하학적 배경 패턴 (골프 코스의 격자무늬)
- AI 칩 모티프를 반복하는 아이콘 시스템

**인터랙션 철학**:
- 마우스 호버 시 네온 그린 글로우 효과
- 스크롤 트리거 시 요소가 부드럽게 나타나는 페이드-인
- 버튼 클릭 시 펄스 애니메이션

**애니메이션 가이드라인**:
- 모든 전환은 300ms 이상의 부드러운 이징 사용
- 스크롤 시 요소별로 스태거 효과 적용 (50ms 간격)
- 호버 상태에서 미묘한 스케일 변화 (1.02배)
- 네온 글로우는 box-shadow로 구현, 부드러운 페이드

**타이포그래피 시스템**:
- Display: Pretentard Bold (900) - 메인 타이틀
- Heading: Pretentard SemiBold (600) - 섹션 제목
- Body: Inter Regular (400) - 본문 텍스트
- Caption: Inter Medium (500) - 보조 정보
- 라인 높이: 1.6 (가독성 우선)

### 아이디어 2: "Organic Tech" (확률: 0.07)

**디자인 무브먼트**: 유기적 형태와 기술의 조화

**핵심 원칙**:
- 곡선과 부드러운 형태를 통한 인간적 접근
- 기술적 정확성과 자연스러움의 균형
- 골프라는 스포츠의 우아함 표현
- 따뜻한 감정과 차가운 기술의 대비

**색상 철학**:
- 배경: Deep Charcoal (#121212) - 밤하늘의 신비로움
- 주요 강조: Neon Green (#39FF14) - 생명의 에너지
- 보조 강조: 라임 그린 (#BFFF00) - 자연의 생동감
- 악센트: 부드러운 보라색 (#8B7FFF) - 프리미엄감

**레이아웃 패러다임**:
- 중앙 정렬이지만 비대칭 요소로 동적성 표현
- 원형과 곡선을 활용한 부드러운 흐름
- 섹션 간 부드러운 곡선 구분선 (SVG divider)
- 콘텐츠 중심의 계층적 구조

**시그니처 요소**:
- 둥근 모서리 카드 (border-radius: 24px)
- 유기적 곡선 배경 패턴
- 골프공 형태의 아이콘 모티프

**인터랙션 철학**:
- 부드러운 호버 효과로 따뜻한 느낌 전달
- 스크롤 시 요소가 자연스럽게 나타남
- 터치 피드백으로 반응성 강조

**애니메이션 가이드라인**:
- 모든 전환은 400ms의 부드러운 이징
- 스크롤 시 요소별로 자연스러운 페이드 및 슬라이드
- 호버 상태에서 부드러운 색상 변화
- 배경 요소의 미묘한 움직임 (parallax)

**타이포그래피 시스템**:
- Display: Pretentard ExtraBold (800) - 메인 타이틀
- Heading: Pretentard SemiBold (600) - 섹션 제목
- Body: Inter Regular (400) - 본문 텍스트
- Caption: Inter Regular (400) - 보조 정보
- 라인 높이: 1.7 (따뜻한 느낌)

## 선택된 디자인: "Cyberpunk Minimalism"

**선택 이유**:
- 요청하신 "매우 현대적이고 테크니컬한 느낌"을 가장 직접적으로 표현
- 네온 그린의 강렬한 대비로 AI 기술의 혁신성 강조
- 여백을 활용한 미니멀한 구성으로 고급스러움 표현
- 비대칭 레이아웃으로 동적이고 현대적인 느낌 전달
- 명확한 계층 구조로 사용자 여정 최적화

**이 디자인의 핵심 특징**:
1. **극도의 단순성**: 불필요한 요소 제거, 본질에 집중
2. **강렬한 색상 대비**: 어두운 배경에 네온 그린의 임팩트
3. **정교한 타이포그래피**: Pretentard와 Inter의 조화
4. **동적 애니메이션**: 스크롤과 호버 시 부드러운 전환
5. **기술적 감각**: 격자무늬 배경, 네온 글로우 효과

## 구현 로드맵

1. **색상 시스템 설정** (index.css)
   - CSS 변수로 네온 그린, 연두색, 차콜 정의
   - 다크 모드 기본값 설정

2. **타이포그래피 설정** (Google Fonts)
   - Pretentard 폰트 임포트 (Bold, SemiBold)
   - Inter 폰트 임포트 (Regular, Medium)

3. **컴포넌트 개발**
   - Hero 섹션: 메인 카피, 서브 카피, CTA 버튼
   - Feature 카드: 3개 기능 카드
   - Pricing 섹션: Free/Pro 플랜
   - CTA 폼: 이메일 입력 및 사전 예약 버튼

4. **애니메이션 구현**
   - Framer Motion을 활용한 스크롤 애니메이션
   - 호버 효과 및 버튼 인터랙션

5. **반응형 디자인**
   - 모바일 우선 접근법
   - 태블릿, 데스크톱 브레이크포인트 최적화

6. **최종 검증**
   - 크로스 브라우저 테스트
   - 성능 최적화
